## Compilação (CMake)

- **Criar o diretório de build**:
    ```bash
    mkdir build
    ```

- **Entrar no diretório de build**:
    ```bash
    cd build
    ```

- **Gerar os arquivos de construção**:
    ```bash
    cmake ..
    ```

- **Compilar o projeto**:
    Linux:
    ```bash
    make
    ```
    Windows:
    ```bash
    cmake --build .
    ```
- **Executar o programa**:
    Linux:
    ```bash
    ./main
    ```
    Windows:
    ```bash
    .\main.exe
    ```
